Implementation

How can we use this system and where will this money come from?

At this stage, our application will be used as a simulation tool; there will be more samples with various cases to show how we could improve the system.

After a while, if it get can enough support and shows that it can make a difference, there are two possible ways to implement it.

Crowd Fund / Investment: Less effective but easy to implement

We can create various crowd funded piggy banks and use these funds to support the organizations based on the results that we collectively decide. Naturally the impact of this option would be limited, however would be an easy start.

Government Taxes / Subsidies: Difficult to achieve but ultimate solution

Push political parties, governments to use this approach as a sort of subsidy program that will be funded through tax system. It would be difficult to achieve this but in order to solve these problems, we believe this should be a society level contract that we all equally contribute.
