Conclusion

The main idea of this project is to show what are the possibilities, to give an idea what we could do to solve our issues. Rest of the details can be debatable.

If we see economic systems as a way of resource distribution, it is clear that the existing system can be improved dramatically. Good news is that the technology is getting better at an accelerated speed, so our chances of addressing and creating new solutions.

Impossible Task

It should be mentioned that we already have two important tools to improve the current system;

  1. Competition between the organizations
  2. Increasing the awareness of the consumers

Both serves for the benefit of the society and even if we don't develop applications like this one, these tools will be improving the system in the long run; we will have better organizations in time and the consumers will have the knowledge to choose the right ones. However this still would be unnecessarily slow process. If an organization has a clear advantage for the society, the system could be modified in such a way that it could support this benefit automatically.

The main issue in the current system is that it leaves the whole responsibility to the consumers; every consumer must know every detail about the products and their organizations before making any single purchase, therefore allocating the resources. The amount of information has be collected and processed by each consumer is simply massive and expecting consumers to fulfill this task is an impractical one.

In today's system, this knowledge gap is closed by marketing; the consumers make their choices not based on facts but based on advertisements, packages and colors. Marketing is a powerful tool and it will always be there but controlling the consumers and shaping the results only based on feelings is not enough for a good resource management.

We believe, the society needs complex systems to be able to manage its resources, in order to become healthier and more productive. Creating such system is a long shot and this project is just a baby step. There are many areas to be improved, functionality, performance, security and so on. It will take time to create a solid application. However, developing a such system could play a vital role on solving some of society's long term issues.
