Introduction

An economic system is responsible for allocating the resources to the organizations, most commonly via the transactions between the organizations and the consumers. In today's system, in order to get more resources, an organization has to;

  • Maximize the price of its products, services, therefore the profit
  • Increase the number of sales
However, organizations have also other features that can be measured and equally important for the society; like responsibilities towards its workers, being environment friendly or even producing healthy products.

Wealth Economy is an attempt to create a collaborative, customizable system to allow the society manage its monetary resources. It aims to improve the system by encouraging the organizations that are more beneficial to the society, using the following steps;

  • Creates Crowd Managed Resource Pool (CMRP) that holds certain amount to be distributed
  • Define Indexes for the organizations which are various parameters, categories that seek benefit to the society
  • Measure the organizations' overall performance on these parameters and allocate CMRP Amount based on the outcome
